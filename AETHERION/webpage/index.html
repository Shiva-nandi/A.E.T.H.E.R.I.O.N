How The System Works

From drone camera pixel to confirmed alert dispatch — 6 stages, all under 200ms end-to-end.

01

Video Capture & Aquatic Preprocessing

4K drone frames captured via OpenCV. AquaticPreprocessor removes sun glare via CLAHE + inpainting, enhances contrast, and corrects camera shake with optical flow stabilization — essential for reliable detection over water.

preprocess.py · AquaticPreprocessor~5ms
02

YOLOv8 Multi-Threat Inference

Preprocessed frames fed into fine-tuned YOLOv8s model with 11 output classes. YOLO_CLASS_MAP converts each class index to a ThreatClass enum. Running at ~12ms on T4 GPU, ~85ms on Jetson Nano, ~18ms on Jetson Orin.

inference.py · MultiThreatDetector12–85ms
03

Multi-Frame Threat Confirmation

ThreatClassifier tracks each detection across frames. Critical threats require multi-frame confirmation (e.g. unconscious needs 15s continuous, shark needs 3s) before escalating — preventing false positives from splashing or glare.

classifier.py · ThreatClassifier~2ms
04

Severity Scoring & Responder Lookup

Confirmed events are scored using THREAT_CONFIG severity matrix. ResponderLookup queries Overpass API (OpenStreetMap) for nearest police station and hospital by GPS, with Hyderabad hardcoded database as fallback for offline demo.

responder_lookup.py · ResponderLookup~50ms (cached)
05

Smart Alert Dispatch

SmartAlertEngine routes to correct channels by severity: Twilio SMS + auto voice call to PS/hospital for CRITICAL, Firebase push to lifeguard app for HIGH/MEDIUM, public broadcast for LOW. 60-second per-class cooldown prevents spam.

smart_alert_engine.py · SmartAlertEngine<5s
06

Autonomous Rescue Navigation

Drone converts victim GPS to NED coordinates and navigates at 5 m/s to victim location. Descends to 1.5m, releases rescue buoy, ascends and continues patrol. Full mission target: under 60 seconds from detection to buoy deployment.

airsim_controller.py · run_rescue_mission()<60s total

# AETHERION — Threat Classifier
# Multi-frame confirmation + severity scoring

class ThreatClassifier:
  def process_detections(self, detections, drone_pos):
    newly_confirmed = []

    for det in detections:
      cfg = THREAT_CONFIG.get(det.threat_class)
      # Gate by confidence threshold
      if det.confidence < cfg["conf_threshold"]:
        continue

      key = f"{det.threat_class.value}_{det.gps}"

      if key in self._active_events:
        event = self._active_events[key]
        event.frame_count += 1
        # Multi-frame confirmation check
        if not event.confirmed:
          confirm_needed = cfg["multiframe_confirm_sec"]
          if event.duration_sec() >= confirm_needed:
            event.confirmed = True
            newly_confirmed.append(event)
      else:
        self._active_events[key] = ThreatEvent(
          threat_class = det.threat_class,
          severity     = cfg["severity"],      # CRITICAL/HIGH/MEDIUM/LOW
          confidence   = det.confidence,
          gps          = det.gps,
          responders   = cfg["responders"],
        )

    return newly_confirmed

# AETHERION — Smart Alert Engine
# Routes alerts to correct responders by severity

class SmartAlertEngine:
  COOLDOWN_SEC = 60

  def dispatch(self, event, is_beach_zone=True):
    # Per-class cooldown
    key = event.threat_class.value
    if time.time() - self._cooldowns.get(key,0) < self.COOLDOWN_SEC:
      return
    self._cooldowns[key] = time.time()

    # Find nearest responders by GPS
    nearest = self.responder_lookup.get_nearest(
      lat=event.gps[0], lon=event.gps[1],
      responder_types=event.responders
    )

    # Route to correct channels
    for r_type, contact in nearest.items():
      if r_type == 'lifeguard':
        self._alert_lifeguard(message, contact, event)
      elif r_type == 'police_station':
        self._alert_ps(message, contact, event)
      elif r_type == 'hospital':
        self._alert_hospital(message, contact, event)
      elif r_type == 'coast_guard':
        self._alert_ps(message, contact, event, 'coast_guard')

  def _alert_ps(self, msg, contact, event, ps_type='police_station'):
    # SMS always
    self._sms(contact['phone'], msg)
    # Voice call for CRITICAL severity
    if event.severity == Severity.CRITICAL:
      self._voice_call(contact['phone'], event)

  def _voice_call(self, to, event):
    twiml = (f"<Response><Say>"
      f"Emergency. {event.threat_class.value} at "
      f"{event.gps[0]:.4f} north. Immediate response."
      f"</Say></Response>")
    self._twilio.calls.create(twiml=twiml,
      from_=self.from_num, to=to)

class ResponderLookup:
  def get_nearest(self, lat, lon, responder_types):
    result = {}
    for r_type in responder_types:
      # Try live Overpass API first
      contact = self._live_lookup(lat, lon, r_type)
      if not contact:
        # Fallback to Hyderabad hardcoded DB
        db = HYDERABAD_RESPONDERS[r_type+"s"]
        contact = min(db, key=lambda x:
          self._haversine(lat, lon, x['lat'], x['lon']))
      result[r_type] = contact
    return result

  def _haversine(self, lat1, lon1, lat2, lon2):
    R = 6371000.0
    d = math.radians
    dlat = d(lat2-lat1); dlon = d(lon2-lon1)
    a = (math.sin(dlat/2)**2 + math.cos(d(lat1))
         * math.cos(d(lat2)) * math.sin(dlon/2)**2)
    return R * 2 * math.atan2(a**.5, (1-a)**.5)

# 11-class YOLO → ThreatClassifier → AlertEngine

YOLO_CLASS_MAP = {
  0:  ThreatClass.NORMAL_SWIM,
  1:  ThreatClass.PANIC_DROWNING,
  2:  ThreatClass.UNCONSCIOUS,
  3:  ThreatClass.SUBMERGED,
  4:  ThreatClass.SHARK_ATTACK,
  5:  ThreatClass.RIP_CURRENT,
  6:  ThreatClass.JELLYFISH_SWARM,
  7:  ThreatClass.HEATSTROKE,
  8:  ThreatClass.FIGHT_ASSAULT,
  9:  ThreatClass.NET_ENTRAPMENT,
  10: ThreatClass.MONSOON_SURGE,
}

class MultiThreatDetector:
  def process_frame(self, frame, victim_gps=None):
    preprocessed = self.preprocessor.process(frame)
    results = self.model(preprocessed, conf=self.conf)[0]

    raw = [Detection(
        threat_class = YOLO_CLASS_MAP[int(b.cls[0])],
        confidence   = float(b.conf[0]),
        bbox         = b.xyxy[0].tolist(),
        gps          = victim_gps or self.drone_gps,
      ) for b in results.boxes]

    # Multi-frame confirmation
    confirmed = self.classifier.process_detections(
      raw, drone_pos=self.drone_gps)

    # Dispatch alerts for confirmed events
    for event in confirmed:
      self.alert_engine.dispatch(event)

    return {'raw_detections': raw,
            'confirmed_events': confirmed}
